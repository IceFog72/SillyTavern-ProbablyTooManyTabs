# More Extensions by IceFog72

Install any of these from the SillyTavern Extension Manager (third-party tab).

---

## SillyTavern-MoreReasoning

Advanced reasoning controls for AI models that support extended thinking. Configure reasoning budgets, toggle thinking per-request, and inspect the model's reasoning process in a dedicated panel.

**Tags:** AI · Reasoning · LLM

[→ View on GitHub](https://github.com/IceFog72/SillyTavern-MoreReasoning)

---

## SillyTavern-PocketTTS-WebSocket

SillyTavern extension for [pocket-tts-openapi](https://github.com/IceFog72/pocket-tts-openapi). Streams TTS audio sentence-by-sentence so playback starts during generation. Features a full player bar with seek, volume, speed, highlight, and playlist controls. Supports voice cloning via `.wav` files served from the backend. Requires pocket-tts-openapi

**Tags:** TTS · Audio · Voice · Streaming

[→ View on GitHub](https://github.com/IceFog72/SillyTavern-PocketTTS-WebSocket)

---

## pocket-tts-openapi

Fast, local, OpenAI-compatible TTS server with voice cloning support powered by Kyutai's Pocket TTS.

**Tags:** TTS · Server · Local · Voice Cloning

[→ View on GitHub](https://github.com/IceFog72/pocket-tts-openapi)

## SillyTavern-SimpleQRBarToggle

Adds a toggle button to show or hide the Quick Replies bar. Minimal and focused — one button, one job. 

**Tags:** UI · Quick Replies · QoL

[→ View on GitHub](https://github.com/IceFog72/SillyTavern-SimpleQRBarToggle)

## SillyTavern-CustomThemeStyleInputs

Adds a UI panel for managing custom CSS properties (variables) directly from within SillyTavern. Edit theme CSS variables with live inputs instead of hand-editing CSS files.

**Tags:** UI · Themes · CSS · Customization

[→ View on GitHub](https://github.com/IceFog72/SillyTavern-CustomThemeStyleInputs)
